To grab the finished source code for this module:

  • Go to this GitHub page: https://github.com/dannyconnell/vue-composition-api-course

  • From the Branch dropdown, choose the branch for this module (e.g module-3 branch for Module 3):

  • Click on the green Code button, then Download (or clone with git if you prefer):

  • Drag the nested folder (vue-composition-api-basics or vue-composition-api-noteballs) into VSCode (or your preferred editor)

  • In the Terminal, run this to install dependencies:

npm install

  • Then run this to launch the app:

npm run dev

  • There will be a "Local" link displayed in your terminal. Open this link in Chrome (e.g. http://localhost:3000/)
